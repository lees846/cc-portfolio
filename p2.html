Happy Man Silhouette

Link to Code Sketch

ID: An orange background with a silhouetted head which's direction and size is changed with the mouse position. The eye, ear, nose, chin, shoulder, head, and mouth shapes change slightly each time it's clicked.

Description

In this project, we were tasked with creating an interactive face generator using the random function, user-defined variables, and mousePressed() and/or keyPressed().

Design Process

Before jumping straight into coding, part of the assignment this week was sketching out our plan for the face, and illustrate the minimum and maximum scale of our design. Initially, I imagined the scale being on a range of human to goblin-like, as shown below:

My main concept was counting on the ability to soften the points of triangles with relation to mouseX and mouseY, however I found that the triangle function doesn’t take arguments for this function, so I decided to change my idea a bit. I maintained the silhouetted look as well as the profile view, but I realized I liked the simplicity and randomness of a sketch that changes the same characters’ features when the mouse is clicked, and I abandoned the goblin idea.

I started by hard-coding values into the sketch to create a base for the generator, and I ended up with something that I could manipulate the scale and direction it faced using my mouse position, however the random features were re-generated every frame (see example sketch below).

Reflection

I enjoyed working on this assignment. I like to see how the challenges posed by coding change my initial concepts and force me in a new and unexpected direction that is often more interesting. I think the result is fun to play around with, and I am now way more comfortable with push(), pop(), translate(), and scale(). These functions are fairly new to me, but I used them frequently in this sketch and I think it was successful.

If I were to continue working on this project, I would probably try to implement keyPressed() as a means of taking a screenshot of the character that was generated. (If I had that function in the sketch, I would make sure to put a note somewhere that indicates how to use the function.) I also would want to try adding some buttons to change the background color, maybe as well as some animation, sound, or other effect when the mouse is pressed to indicate an action has occurred.
